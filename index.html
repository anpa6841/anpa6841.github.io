    • File Integrity Monitoring (FIM) - Platform Agnostic

  • A robust multithreaded FIM system that continuously checks for file changes.
  • Implemented a hash-checking mechanism using SHA256 algorithm to track the integrity of files over time. When a file is first monitored, its hash is computed and stored in an "integrity database" which is a simple file.
  • Whene a file is modified, its hash is recomputed and compared to the stored hash value. If it differs, an alert is triggered.
